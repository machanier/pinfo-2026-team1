# Frontend

This folder contains the frontend application of the **UNIGEvents** project.

The frontend is a Single Page Application (SPA) built with **React**.

---

## Responsibilities

- User interface
- Event browsing and visualization
- User interactions
- Sending HTTP requests to the backend REST API
- Rendering data returned by the backend

---

## Tech Stack

- Language: JavaScript
- Framework: React
- Package manager: npm
